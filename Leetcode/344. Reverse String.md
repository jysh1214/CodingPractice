# [344. Reverse String](https://leetcode.com/problems/reverse-string/)

```c++
class Solution {
public:
    void reverseString(vector<char>& s) {
        for (int i = 0; i < (int)s.size() / 2; ++i) {
            char tmp = s[i];
            s[i] = s[s.size() - 1  - i];
            s[s.size() - 1  - i] = tmp;
        }
    }
};
```
